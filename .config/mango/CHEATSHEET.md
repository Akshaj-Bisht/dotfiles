# Mango Cheatsheet

## Apps
`Super+Return` foot ôñ `Super+B` browser ôñ `Super+E` thunar

## Noctalia
`Super+Space` launcher ôñ `Super+S` control center ôñ `Super+V` clipboard
`Super+Esc` session ôñ `Super+Ctrl+L` lock ôñ `Super+Ctrl+,` settings
`Super+Shift+T` theme ôñ `Super+Shift+N` DND

## Window
`Super+Q` close ôñ `Super+M` quit ôñ `Super+F` maximize
`Super+Shift+F` fullscreen ôñ `Super+T` float ôñ `Super+O` overview

## Navigation
`Super+H/J/K/L` focus ôñ `Super+Tab` last window
`Super+Shift+H/J/K/L` move window

## Tags
`Super+1..9` go ôñ `Super+Shift+1..9` move window

## Layouts
`Super+N` cycle ôñ `Ctrl+Super+I` tile ôñ `Ctrl+Super+O` scroller
`Ctrl+Super+D` dwindle ôñ `Ctrl+Super+G` grid ôñ `Ctrl+Super+M` monocle

## Scratchpad
`` Super+` `` toggle terminal

## Reload
`Super+R` reload config
